# Goormpen

## 서비스 링크

[Goormpen](http://ec2-15-165-114-166.ap-northeast-2.compute.amazonaws.com/)

## 소개
- 리액트와 스프링을 연동해 배포하였습니다.
- 웹소켓을 이용한 실시간 채팅 서비스입니다.

## 프로젝트 정보

- React, sockjs-client, stompjs
- Spring boot, Websocket
- AWS EC2

## 기능

- 전체 채팅 방
- DM
